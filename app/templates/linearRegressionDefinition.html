Linear Regression - Basic Concepts

What is Linear Regression?

Linear regression is a supervised machine learning algorithm used to model the relationship between a dependent variable and an independent variable. The model predicts continuous values by fitting a straight line to the data.

Independent Variable (X)

The independent variable represents the input data used to make predictions. This variable influences the dependent variable.

Example:
  • Study Hours
  • Advertising Budget
  • Temperature
Dependent Variable (Y)

The dependent variable is the value that we want to predict. It depends on the independent variable.

Example:
  • Final Grade
  • Sales
  • Energy Consumption
Training Data

Training data consists of known input and output values used to train the linear regression model.

Regression Equation

Y = mX + b

Where:

  • Y = Dependent variable
  • X = Independent variable
  • m = Slope
  • b = Intercept
Slope (m)

The slope represents how much the dependent variable changes when the independent variable increases by one unit.

Example:

If slope = 0.5, then each additional study hour increases the grade by 0.5

Intercept (b)

The intercept represents the predicted value when X equals zero. It is the starting point of the regression line.

Example:

If intercept = 2.0, then the predicted grade is 2.0 when study hours are zero

Example Interpretation

If a student studies more hours, the predicted grade increases. Linear regression finds the best line that describes this relationship.
