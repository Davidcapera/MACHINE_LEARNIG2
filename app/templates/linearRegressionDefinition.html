Linear Regression — Basic Concepts

What is Linear Regression?

Linear Regression is a supervised machine learning algorithm used to model the relationship between independent variables and a dependent variable. The goal is to predict continuous numerical values using a best-fit line.

Independent and Dependent Variables

Linear regression uses independent variables (features) to predict a dependent variable (target).

  • X → Independent Variable (Input)
  • Y → Dependent Variable (Output)
Example:
  • Study Hours → Independent Variable
  • Final Grade → Dependent Variable
Training Data

Training data consists of input and output values used to train the linear regression model. The algorithm learns the relationship between variables using this dataset.

Study Hours Final Grade
2 2.5
4 3.0
6 3.5
8 4.0
Regression Line

The regression line represents the best linear relationship between input and output values.

Regression Equation

Y = mX + b

  • Y = Dependent variable
  • X = Independent variable
  • m = Slope
  • b = Intercept
Slope (m)

The slope represents how much the dependent variable changes when the independent variable increases.

Example:

If slope = 0.5, each additional study hour increases the grade by 0.5

Intercept (b)

The intercept represents the predicted value when X equals zero. It is the starting point of the regression line.

Example:

If intercept = 2.0, predicted grade is 2.0 when study hours = 0

Example Application

Linear regression can be used to predict student grades based on study hours. As study time increases, the predicted grade also increases.
