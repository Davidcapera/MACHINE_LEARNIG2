Linear Regression — Basic Concepts

What is Linear Regression?

Linear Regression is a supervised machine learning algorithm used to model the relationship between independent variables and a dependent variable. The goal is to predict continuous numerical values using a best-fit line.

Independent and Dependent Variables

Linear regression uses independent variables to predict a dependent variable.

  • X → Independent Variable
  • Y → Dependent Variable
