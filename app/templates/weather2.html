Case Use Overview
Title

Weather Condition Prediction Using a Decision Tree

Problem Description

Knowing the weather in advance is essential for planning daily activities, agriculture, transportation, and disaster prevention. Traditional forecasting systems require complex meteorological data and expensive infrastructure. This case demonstrates how a simple Machine Learning model can classify weather conditions Rainy, Sunny, Very Sunny, or Cloudy based on only two easily measurable variables: temperature and humidity.

Type of Machine Learning
Supervised Learning Classification Decision Tree
Input Features
  • 1
    Temperature (°C)

    Ambient temperature measured in Celsius degrees. Range used in training: 14 °C and 40 °C.

  • 2
    Humidity (%)

    Relative humidity percentage in the atmosphere. Range used in training: 24 % and 44 %.

Expected Output

The model returns one of the following weather condition labels:

Cloudy Rainy Sunny Very Sunny
Real-World Relevance

Accurate weather classification enables smarter decisions in sectors such as agriculture (irrigation scheduling), logistics (route planning), tourism (activity recommendations), and civil protection (early warnings). This lightweight model proves that even a small, well-structured dataset can produce actionable predictions, making it ideal for resource-constrained environments or educational contexts.
