Supervised Machine Learning model for binary classification with performance analysis

Model Explanation

The SGDClassifier (Stochastic Gradient Descent Classifier) is a supervised machine learning algorithm used for classification problems. It works by iteratively updating model parameters using small subsets of data, making it efficient for large-scale datasets.

In this application, the model is used to perform a binary classification task, predicting whether rain is expected or not based on environmental variables such as temperature, humidity, wind speed, and pressure.

Dataset Description

The dataset used is the Iris dataset, obtained from the Scikit-learn library.

Source: Scikit-learn Iris Dataset

The dataset contains measurements of iris flowers and was adapted into a binary classification problem to simulate a rain prediction scenario.
