Classification Model — Basic Concepts

What is a Classification Model?

A classification model is a supervised machine learning algorithm used to categorize input data into predefined classes.

The model learns patterns from labeled training data and assigns new observations to the most probable category.

Classification Types
Binary Classification

Two possible classes

  • Yes / No
  • Spam / Not Spam
Multi-class Classification
  • Sunny
  • Rainy
  • Cloudy
Multi-label Classification
  • Multiple outputs
  • Image tagging
Input Variables
  • Temperature
  • Humidity
  • Wind Speed
  • Age
  • Income
  • Pressure
Output Classes

The classification model assigns input data to predefined categories.

  • Sunny
  • Rainy
  • Cloudy
Decision Boundary

Classification models separate data into regions using decision boundaries. These boundaries divide the feature space into class regions.

Example Application

A classification model can predict weather conditions based on temperature and humidity.
