Rain Prediction — SGDClassifier

Binary classification using Stochastic Gradient Descent

1 · Problem Description

This model predicts whether it will rain tomorrow using the SGDClassifier algorithm. The classifier learns a linear decision boundary using Stochastic Gradient Descent.

Type of Machine Learning

Supervised Learning Binary Classification SGDClassifier

Input Features

  • Min Temperature (°C)
  • Max Temperature (°C)
  • Rainfall (mm)
  • Humidity (%)

Expected Output

Rain No Rain
Rain prediction supports decision-making in agriculture, logistics and weather planning.
