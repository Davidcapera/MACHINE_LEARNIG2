Logistic Regression — Basic Concepts

What is Logistic Regression?

Logistic Regression is a supervised machine learning algorithm used for classification problems. Unlike linear regression, logistic regression predicts categorical outcomes by estimating probabilities using the sigmoid function.

The model outputs values between 0 and 1, which represent the probability of belonging to a specific class.

Binary Classification

Logistic regression is primarily used for binary classification tasks where the output variable has two possible categories.

  • Spam / Not Spam
  • Fraud / Not Fraud
  • Pass / Fail
  • Yes / No
  • Customer / Not Customer
  • Disease / Not Disease
  • Approved / Rejected
  • Click / Not Click
Sigmoid Function

Logistic regression uses the sigmoid function to convert linear predictions into probabilities between 0 and 1.

P(Y=1) = 1 / (1 + e^-x)

If the probability is greater than 0.5, the model predicts class 1. Otherwise, it predicts class 0.

Independent and Dependent Variables
Independent Variables (X)
  • Age
  • Income
  • Temperature
  • Study Hours
  • Marketing Budget
Dependent Variable (Y)
  • 0 → Negative Class
  • 1 → Positive Class
Decision Boundary

Logistic regression creates a decision boundary that separates different classes based on probability values.

The threshold is typically 0.5, but it can be adjusted depending on the problem.

Example Application

Logistic regression can predict whether a student will pass based on study hours. The model calculates the probability of passing and assigns a class.
